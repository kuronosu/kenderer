# Sample assets

Generated by a dev-only tool (the `_gen` directory is excluded from
`go build ./...`):

```
go run examples/_gen/main.go
```

- `torus.glb` — textured donut, texture **embedded** in the GLB (self-contained).
- `sphere.obj` + `sphere.mtl` + `sphere_uv.png` — textured UV sphere.

Both use a UV-checker texture (red increases with U, green with V) so the texture
orientation is easy to read, and both are curved so the per-object **smooth
shading** (`scene.Object.Smooth`) is visible.

View them in the interactive viewer (needs the `sdl` build tag):

```
go run -tags sdl ./cmd/viewer -model examples/torus.glb
go run -tags sdl ./cmd/viewer -model examples/sphere.obj
```

Drag to orbit, scroll to zoom, middle-drag (or shift+drag) to pan. The camera
auto-frames the model on load.

`preview_torus.png` / `preview_sphere.png` are offscreen renders the generator
writes for a quick eyeball (gitignored).
